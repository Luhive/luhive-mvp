# 10-07-2026

## Event Hosted By — WhatsApp join for selected communities

Some communities (starting with Innovation Wednesdays / Enhance) can send people to their WhatsApp group from the event page **Hosted By** button instead of the usual Luhive Join/Joined flow.

- On the event page, if a community opts in and has a WhatsApp link saved on their profile, the Hosted By button shows **Join WhatsApp** and opens that group
- Everyone else keeps the normal Join / Joined behavior
- Event registration still adds people to the Luhive community as before
- Community public page Join is unchanged

**Ops note:** enable with community `settings.event_hosted_by_join = "whatsapp"` and a WhatsApp URL in `social_links` (dashboard social links). Example:

```sql
update communities
set settings = coalesce(settings, '{}'::jsonb) || '{"event_hosted_by_join":"whatsapp"}'::jsonb
where slug = 'innovation-wednesdays';
```
