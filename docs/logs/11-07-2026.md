# 11-07-2026

## Dashboard settings — turn off registration emails

Admins can now control whether they get an email every time someone registers for their community’s events. Useful when registration volume is high and the inbox gets noisy.

- **Settings (dashboard):** new Settings item in the sidebar opens a Notifications page
- **Toggle:** turn **New registration emails** on or off for yourself only — other admins keep their own preference
- **Default:** emails stay on until you turn them off
- **Scope:** only registration alerts for that community; other admin emails are unchanged

## Event sharing — custom link preview image

Sharing an event link (WhatsApp, LinkedIn, X, Slack, iMessage, etc.) now shows a branded Luhive preview card instead of just the raw cover photo — similar to how Luma previews look.

- **What people see:** the community's logo and name, the event title in Luhive orange, the event cover as a rounded card, and the Luhive logo — laid out to match the design
- **Where it appears:** automatically on any shared public event link; nothing to configure
- **Fallbacks:** if the event has no cover it uses the community logo, and if a link can't be matched to an event it falls back to the default Luhive image, so previews never break
- **Freshness:** the preview updates when the event's title, cover, or host changes, and is cached aggressively otherwise so shared links load fast
