# 06-07-2026

## Sidebar — quick access to your profile

From the dashboard sidebar, you can now open your personal profile directly from the user menu.

- Click your name/avatar at the bottom of the sidebar to open the menu
- Tap the user info at the top of that menu (avatar, name, email) to go to your profile page
- Log out option is unchanged

## Community profile — Discord link

Communities can now add a Discord invite/link alongside their other social accounts.

- **Edit profile (dashboard):** new Discord field under Social Media accounts, with the Discord icon
- **Public community page:** Discord link appears with the other social buttons when one is saved
- Discord icon is shown in grayscale so it matches the look of the other social icons

## Event registration — sign in with Google

Guests can now register for an event using Google, the same way they can join a community — without going through email OTP first.

- **Register overlay:** new “Continue with Google” option on the first step, alongside email registration
- **Simple events (no extra questions):** after Google sign-in, registration completes automatically and the user lands back on the event page as registered
- **Events with custom questions:** Google sign-in first, then the user is sent back to the register overlay to answer the required questions before registration is finalised
- **Join community:** the existing checkbox still applies — if unchecked before Google, the user registers for the event but does not join the community
- **Approval-required events:** registration goes to pending approval; the user gets the same pending flow as email registration
- **Already registered:** signing in again does not create a duplicate registration
- **Invites:** invitees who arrive through the invite flow still complete registration via the invite path (custom questions only) — they do not see the Google option, since their email is already verified through the invite
